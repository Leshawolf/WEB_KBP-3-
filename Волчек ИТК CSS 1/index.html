Что такое CMS

        CMS - "система управления контентом"
        (движок) - написания PHP-программистами основа
        для сайта, с помощью которой вы сможете
        управлять сайтом (добавить контент, менять пункты
        меню и т.п). не зная HTML и CSS.

            Однако, для того чтобы сделать сайт с
        помощью CMS потребуются услуги и
        программиста, и дизайнера, и верстальщика. И
        капиталовложения.
        
Какие бывают cms

            Бывают различные системы управления
        клиентом: для интернет-магазинов, для блогов,
        для форумов и т.д.
        
Примеры cms

            Примеры популярных CMS: Joomia, WordPress
        (для блогов), PhpBB (для форумов).

            CMS-ки бывают платные и бесплатные.
